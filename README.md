## ⚙️ How to use the software

### Downloading Videos
Open the software and navigate to the "Downloader" tab. Paste the URL of your Douyin video collection into the input field. Press the start button. The software saves these files to your selected output folder.

### Generating Subtitles
Switch to the "Subtitle" tab. Select the video file you processed earlier. Choose the language settings for your audio. Click the "Transcribe" button to use the local Whisper engine. This process happens entirely on your machine.

### Using Gemini Translation
Go to the "Settings" tab to enter your Gemini API key. Return to the "Translator" tab to upload your generated subtitle file. Select your target language. Click the translate button to generate the new file.
