🌟 Overview
Galaxy Ultimate Studio is a comprehensive professional solution designed to automate the entire video production workflow. Whether you are targeting Dailymotion, Facebook, or short-video platforms, this system handles everything from bulk downloading to high-quality localized output using cutting-edge AI.

🛠️ Triple-Power Modules
1. 📥 Bulk Douyin Downloader V2.0
Mass Downloader: Supports bulk processing via individual links or .txt list files.

Stealth Mode: Automated bot-detection bypass and QR code removal.

HD Quality: Extracts original HD video and audio streams, automatically merging them via FFmpeg.

2. 🎧 AI Subtitle Extractor (Offline)
GPU Accelerated: Fully optimized for NVIDIA CUDA (Recommended: GTX 1070 or higher) for lightning-fast processing.

Whisper AI Integration: Utilizes OpenAI's Whisper models for high-precision speech-to-text transcription.

100% Offline: No API costs, total privacy, and works without an internet connection.

3. 🌍 Gemini API Global Translator V3.0
Contextual Intelligence: Direct integration with Google Gemini API for human-like translation.

Global Language Support: Localize content into English, Vietnamese, Chinese, Japanese, Korean, Thai, and 100+ other languages.

SRT Integrity: Maintains perfect timestamp formatting and synchronization.

⚡ System Requirements
OS: Windows 10/11 (64-bit).

Hardware: NVIDIA GPU with CUDA support is highly recommended for AI modules.

Dependencies: Python 3.10+, FFmpeg, Aria2c.
